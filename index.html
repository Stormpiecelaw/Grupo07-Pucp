Proyecto de Análisis de Datos

Dashboard desarrollado con Power BI

Introducción

Este proyecto presenta el desarrollo de un dashboard interactivo utilizando Power BI con el objetivo de analizar datos y generar visualizaciones que permitan identificar patrones relevantes.

Objetivo del Proyecto

Transformar datos en información útil mediante herramientas de Business Intelligence que faciliten la interpretación de los datos y la toma de decisiones basada en evidencia.

Metodología

  • Limpieza y preparación de datos
  • Análisis exploratorio
  • Visualización mediante Power BI
  • Construcción de dashboard interactivo
